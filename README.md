01
目的
QRコードを読み取って
食べ物の基本情報を表示する
02
動機
普段の生活で食品の原材料やアレルギー情報をすぐに確認できれば、
アレルギーのある方や障害のある方が、より安心して食品を選ぶことが
でき、生活の負担を少しでも軽減できると考えたため。
03
内容
商品名、原材料、
アレルギー源(28種類)
【アレルギー：特定原材料とそれに準ずるものを分ける】、
保存方法
04
色とフォント、その他
色：#00ff66 (メイン)、#e0e0e0、#0d0f12、#00ff66
フォント：Shippori Mincho、serif
googlefontより : Shippori Mincho
05
改善点・追加予定のもの
・音声読み上げ機能・履歴機能・検索機能・賞味期限,消費期限の表示
・栄養成分表示・多言語対応・バーコード対応
